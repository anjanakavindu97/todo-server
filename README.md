# TODO Application Server

## Recomended node version

v22.20.0

## Setup database

start XAMPP server 

import given "todo" table to the database

## Create env file

PORT=3000

NODE_ENV=development

DB_HOST=

DB_USER=

DB_PASSWORD=

DB_NAME=

## Install packages
```js
npm install
```

## Run server
```js
npm run dev
```
